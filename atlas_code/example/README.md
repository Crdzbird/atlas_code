# atlas_code example

A searchable country browser demonstrating the atlas_code plugin:

* Device country card with flag, codes, dial code and currency.
* Full country list with OS localized names, sorted alphabetically.
* Diacritic insensitive search over names, codes and dial codes.
* Automatic refresh when the system language or region changes.
* English fallback indicator on platforms without OS locale data.

Run it with `flutter run` from this directory on any supported platform.
